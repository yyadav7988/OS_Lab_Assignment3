- `cpu_scheduling.py`  
  - Priority Scheduling (non-preemptive) with Gantt chart  
  - Round Robin Scheduling with Gantt chart  
  - Calculates waiting time and turnaround time

- `file_allocation_sequential.py`  
  - Simulation of **Sequential File Allocation**

- `file_allocation_indexed.py`  
  - Simulation of **Indexed File Allocation**

- `memory_allocation_contiguous.py`  
  - **First Fit**, **Best Fit**, **Worst Fit** contiguous memory allocation

- `mft_mvt.py`  
  - Simulation of **MFT (Fixed Partition)** and **MVT (Variable Partition)** memory management
